A. Tabbed Panels (Recommended)

Conversation, Terminal, Events, Diff, Todos as tabs. One panel visible at a time -- full height. DAG pipeline inline in the header bar. Chat input pinned at bottom of Conversation tab. Clean, focused, like Cursor's agent view. Keyboard shortcuts to switch tabs (1-5).

B. Split Panes (Conversation + Side Panel)

Conversation always visible on the left. Terminal/Events/Diff in a tabbed side panel on the right. Like VS Code's editor + panel. See the agent's thinking AND its output simultaneously. Side panel is resizable and collapsible. More information density but splits horizontal space.

Key decisions baked in

  • DAG pipeline always visible in the session header (not a separate tab)
  • Chat input pinned at bottom of conversation area
  • Tool calls rendered as collapsible blocks (like Cursor 3)
  • Session list on the left (from the icon-rail master-detail layout)
