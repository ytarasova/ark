B. Session Grid

Compact KPI bar + session cards in a grid layout. Each card shows status, summary, mini pipeline, agent, and cost. More visual, like Windsurf's Kanban. Better for fleet overview when running many agents simultaneously.

These aren't mutually exclusive -- we could offer both as views (list vs grid toggle), like how Linear lets you switch between list/board/timeline. The question is which is the default experience.
