A. Aggressive -- 6 items (Recommended)

Sessions = Sessions + History (tab/filter for past sessions)
Resources = Agents + Flows + Tools + Runtimes (tabbed)
Compute = Compute (stays -- distinct enough)
Knowledge = Memory + Search (knowledge graph view)
Costs = Costs (stays -- key differentiator)
Settings = Settings + Schedules + Config + Router
No Dashboard page. Sessions IS the home screen. The status chips in the header give you the overview. Cmd+K for everything else.

B. Moderate -- 8 items

Dashboard -- overview home
Sessions -- active + history (merged)
Agents -- agent definitions
Flows -- flow definitions
Compute -- compute fleet
Knowledge -- memory + search
Costs -- spending dashboard
Settings -- config + schedules
Keeps Dashboard and splits Agents/Flows. More items but less nesting. Each icon rail slot is clearly distinct.
