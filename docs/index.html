Ark treats coding agents as first-class citizens — with isolated compute, shared memory, and a control plane that's actually opinionated about the SDLC.

DAG-based SDLC flows

Compose plan → spec → implement → test → ship nodes as code. Branch, retry, gate on human review.

11 compute providers

local, docker, devcontainer, firecracker, EC2, E2B, k8s, kata-containers. Same API, any substrate.

Multi-runtime

Claude Code, Codex, Gemini CLI, Goose. Swap runtimes per node, or A/B test two on the same task.

Knowledge graph

Code-graph extraction lets agents query call sites, type dependencies, and prior diffs — not just grep.

LLM router

OpenAI-compatible proxy with 3 policies and circuit breakers. Route by cost, latency, or model affinity.

Multi-tenant control plane

Tenants, tokens, audit — ready for teams. Postgres-backed for production, SQLite for local.
