ark

Autonomous agent ecosystem.
Orchestrate AI coding agents through DAG-based SDLC flows with 11 compute providers.

DAG flow engine knowledge graph LLM router 5 runtimes 12 agents 11 compute providers compute templates TensorZero gateway multi-tenant control plane billing modes TUI + web + desktop fan-out / auto-join cost tracking Helm chart remote client mode

Install

$ curl -fsSL https://ytarasova.github.io/ark/install.sh | bash
# Pin to a specific version
$ curl -fsSL https://ytarasova.github.io/ark/install.sh | ARK_VERSION=v0.1.0 bash

Quick Start

$ ark tui # terminal dashboard
$ ark exec --repo . --summary "Fix auth bug"
$ ark search "authentication"
Self-contained: macOS or Linux (arm64/x64). The installer ships a tarball that bundles ark + tmux + codegraph + tensorzero. No system dependencies beyond git.

What you get

⚙

DAG Flow Engine

YAML-defined SDLC pipelines: intake, plan, audit, execute, verify, close, retro. Fan-out, auto-join, verification gates.

⌨

TUI + Web + Desktop

Terminal UI (9 tabs), web dashboard with cost charts, and Electron desktop app. Live agent output.

⚡

11 Compute Providers

Local, Docker, DevContainer, Firecracker, EC2, EC2+Docker/DevContainer/Firecracker, E2B sandboxes, Kubernetes, and K8s with Kata Containers.

🔍

Knowledge Graph

Unified knowledge across codebase, sessions, memories, and learnings. ops-codegraph indexing, 33 languages via tree-sitter.

🔒

Multi-Runtime + Billing Modes

4 runtimes (Claude, Claude Max, Codex, Gemini). 12 specialized agents. API or subscription billing, per-runtime cost tracking.

🔧

Compute Templates

Named presets for compute creation. One YAML entry spins up any provider with pinned size/region/image.

🚀

TensorZero Gateway

Optional Rust-based OpenAI-compatible gateway. Auto-started by the router for circuit breakers, fallbacks, and observability.

📨

Multi-Tenant Control Plane

Hosted mode with API keys, tenant policies, integration enforcement (router_required, auto_index_required), worker registry, Helm chart.
