Quick Start

Get up and running in under 2 minutes.

Prerequisites

  • macOS (arm64 or x64) or Linux (arm64 or x64)
  • git (for worktree-based session isolation)

The install script downloads a self-contained tarball that bundles everything Ark needs: the Ark binary (with Bun runtime compiled in), tmux, the codegraph indexer, and optionally tensorzero-gateway. No package manager, no runtime installs.

If you're installing from source via make install, you'll need Bun and tmux installed on your system.

Install

$ curl -fsSL https://ytarasova.github.io/ark/install.sh | bash

Or clone and install manually:

$ git clone https://github.com/ytarasova/ark.git
$ cd ark
$ make install

Verify Your Setup

$ ark doctor

Checks that Bun, tmux, git, and Claude CLI are installed and working. Run this after a fresh install or when something seems off.

Initialize a Repository

$ cd your-project
$ ark init

Interactive setup wizard: verifies prerequisites, checks Claude authentication, and creates an .ark.yaml config file with your preferred defaults (flow, agent, compute).

Your First Session

1. Start the server daemon and launch the Web UI

$ ark server daemon start --detach
$ ark web

Or install the Electron desktop app and open Ark from your Applications/Start menu.

2. Create a Session

Click New Session. Fill in:

  • Name: A short description (e.g., "add-auth")
  • Repo: Path to your repository
  • Compute: local (default)
  • Flow: bare for a single agent, default for full SDLC pipeline

3. Dispatch

Click Dispatch to launch the agent. The agent runs in a tmux session and starts working.

4. Monitor

  • Watch live output in the Session Detail pane
  • Use the embedded terminal to attach to the agent
  • Use the Chat panel to send messages to the running agent

5. Headless Mode (CI/CD)

$ ark exec --repo . --summary "Fix the auth bug" --flow bare
$ ark exec --repo . --summary "Add tests" --timeout 600 --output json
Note: The bare flow runs a single worker agent with no planning or review stages. Use default for the full planner → implementer → reviewer pipeline.
