CLI reference

Every ark command, in one place. Flags shown with short + long forms; env vars in the Config page.

Sessions

CommandDescription
ark new <prompt>Spawn a new session in the current repo
ark lsList sessions (add --all for stopped)
ark attach <id>Attach to a session's tmux window
ark tail <id>Stream the event log
ark pause <id>Pause without releasing compute
ark stop <id>Terminate session & release compute
ark branch <id>@<event>Fork from a checkpoint

Flows

ark run <flow> <input>Execute a flow
ark flow lsList available flows
ark flow show <name>Print a flow's DAG
ark approve <flow>#<node>Release a human gate

Dev & infra

ark devStart API + web dashboard + daemon
ark conductor start|stop|statusManage the arkd agent daemon
ark router statsLLM backend health & latency
ark gcPrune stopped sessions older than 7d

Global flags

--compute <target>Override compute provider for this run
--runtime <name>Agent runtime (claude, codex, gemini, goose)
--router <policy>LLM router policy
--tenant <id>Tenant scope (control-plane only)
--jsonMachine-readable output
-v, --verboseShow debug logs
