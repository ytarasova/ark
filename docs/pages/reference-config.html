Config reference

Ark resolves config from five layers (highest precedence first): programmatic > CLI flags > ARK_* env vars > ~/.ark/config.yaml > profile defaults.

Profiles

Set with ARK_PROFILE or auto-detected:

  • local — default, SQLite storage, no auth
  • control-plane — Postgres, required auth, multi-tenant
  • test — ephemeral arkDir + random ports per call

YAML format

# top-level defaults apply to every profile
ports:
  conductor: 19100
  arkd: 19300
channels:
  basePort: 19200
  range: 10000
observability:
  logLevel: info

# profile overlays merge on top of the top level
profiles:
  control-plane:
    auth:
      requireToken: true
  test: {}
