# Design research

Reference screenshots used while iterating on Ark's web UI.

## `ark/`

Snapshots of Ark's own UI across themes (arctic-slate, midnight-circuit,
warm-obsidian) and evolutions of the v0-authored prototype. Named by theme +
surface (cmdk, light/dark, verify, etc.) so a diff between two runs is
skimmable.

## `references/`

External products we pull from for visual + interaction patterns. Not an
endorsement; each row is a specific thing we liked at capture time:

- **Cursor** -- autocomplete drawer, agent tab layout.
- **Datadog** -- dashboard density, live metric tiles.
- **Devin** -- autonomous-agent product framing, enterprise page.
- **GitHub Copilot** -- agent blog, chat panel.
- **Linear** -- issue list density, method docs, feature page tone.
- **Ona** -- software factory UX, background agents list.
- **Raycast** -- cmdk interaction, Pro page layout.
- **Replit** -- AI assist UI, scroll narrative.
- **v0 / Vercel** -- Geist color system, observability layout.
- **Windsurf** -- 2.0 launch blog, agent demo scrolls.

## `v0-prototype/`

Captures of the v0-generated prototype we evolved into the current dashboard.
