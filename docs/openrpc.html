OpenRPC Schema

Machine-readable JSON-RPC method definitions with request/response schemas.

The full schema is available as openrpc.json -- an OpenRPC 1.3 document generated from the Zod schemas in packages/protocol/rpc-schemas.ts. Use it with any OpenRPC-compatible tooling (playground, mock server, client generators).

Run bun run scripts/generate-openrpc.ts to regenerate after adding new methods.
